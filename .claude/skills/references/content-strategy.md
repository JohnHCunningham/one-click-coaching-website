# OCC Content Strategy — LinkedIn System

## The Five Pillars

| Pillar | Purpose | Attracts |
|--------|---------|---------|
| Neuroscience / Data | Credibility. You're the expert. | "What's the science behind this?" |
| Story + Insight | Emotion. They see themselves. | "This is exactly my team." |
| Contrarian Take | Authority. You challenge the room. | "I've never thought of it that way." |
| Thought Leadership | Vision. You see what's coming. | "Where is this going?" |
| Social Proof / Outcome | Evidence. You've done it. | "Can you show me the numbers?" |

## Weekly Rotation (3x/week)

| Monday | Wednesday | Friday |
|--------|-----------|--------|
| Neuroscience or Thought Leadership | Story + Insight | Contrarian or Social Proof |

Social proof weaves into whichever post it fits — it doesn't need its own slot every week.

## Round Structure

- **Round 1** — Names the problem. Establishes the gap. Creates the question.
- **Round 2** — Shows the mechanism. Explains why the problem persists.
- **Round 3** — Shows the solution in action. Names OCC explicitly.

Never pitch the product in Round 1. Never.

## Hook Principles

Curiosity through restraint. Four hook styles that work:

1. **The Number** — "72 hours." / "37 calls. One pattern."
2. **The Incomplete Statement** — "He'd been through every program they offered."
3. **The Provocation** — "More coaching isn't the problem."
4. **The Scene** — Drop into a specific moment. No context. Just presence.

## Post Architecture

```
Hook (1–2 lines — creates the tension)
↓
Setup (2–3 lines — the situation most readers recognize)
↓
Pivot (1 line — the turn. Often starts with "But." or "Except.")
↓
Insight (2–3 lines — the truth they hadn't named)
↓
Landing line (1 line — the aphorism. Standalone. Memorable.)
↓
CTA (comment-only, low friction, curiosity-extending)
```

## CTA System

Never: links, DMs, "book a call", "check out my product."

Always: "Drop a comment if you want to see what that looks like."
Or: "Curious if others have seen the same thing — drop a comment."
Or: "Happy to walk you through it if you want to see it live."

## Comment-to-Call Conversion

The sequence: Post creates curiosity → comment asks question → public reply deepens
the question → private conversation opens → discovery call booked.

**The rule:** Answer every comment publicly in a way that makes the next question inevitable.
Never answer fully. Leave the mechanism unexplained. End with a low-friction live invite.

**Response template structure:**
1. Affirm the question (1 line — shows you read it)
2. Partial answer — enough to be useful, not enough to be complete (2–3 lines)
3. The live invite — "Happy to show you what that looks like in practice."

**Question → Response mapping:**

| Question | Response angle |
|----------|---------------|
| "What does the scoring output look like?" | Name the 8 components, tease the non-repetition engine, invite them to see it live |
| "How is this different from Gong?" | Gong is conversation intelligence for managers. OCC is coaching automation for reps. Different problem, different buyer. |
| "What does it cost?" | Depends on team size. Starter tier covers teams up to 10 reps. Worth a 20-minute conversation to see if it fits. |
| "Does it work with [tool]?" | Name what's built (HubSpot, Fathom, Aircall, Fireflies). Invite them to confirm their stack on a quick call. |
| "Is it only Sandler?" | Sandler is the fully built methodology. Others are on the roadmap. If your team uses Sandler, it's the right fit now. |
